Dashcoin (DSC) is an automated anonymous cryptocurrency

Jun 21st, 2014

Dashcoin released

Today is the first day of the life of DashCoin (DSC) – the first automated cryptocurrency using the CryptoNote protocol. The primary goal of the project is to create a fully automated mutable copy of Bytecoin.
